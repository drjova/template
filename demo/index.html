Template template.js

Let you parse a json object. Demo

Options

Options can be passed via data attributes data-{name}

Name Type Markup Description
data-replace string <div data-replace="a key"></div> It replace a single json key with its value
data-repeat string <ul data-repeat="an array">
  <li><p data-replace="a key"></p></li>
</ul> 		It loops throw a json object and parse the inline template
data-logic string <div data-replace="!a key or a key">
    It's empty :(
  or
     It's not empty :)
</div> 		Use ! if it is false or empty otherwise just the key
data-toggle string <a href="#" data-toggle="#test">Toggle that</a>
<div id="test">I will be visible or hidden </div> 		a jquery valid selector to bind the element that you want

Usage

$('#test').template({ 'firstname' : 'A firstname','lastname'  : 'A lastname','positions' : [{'title':'A position title', 'year': 'A year'},{'title':'A position title', 'year': 'A year'},{'title':'A position title', 'year': 'A year'},{'title':'A position title', 'year': 'A year'}],'age' : false})
<div id="test"><p data-replace="firstname"></p> <ul data-repeat="positions"> <li><span data-replace="title"></span></li></ul><p data-logic="!age">Guess what no age!</p></div>
